# 计算机毕设选题-小程序
      小程序类毕设需要搭配后台管理系统，配合小程序前端组件库进行开发，大大增加了创新性和ui的一致性。小程序实现较简单，需要掌握的技术栈也不多，适合专业知识没那么丰富的同学。在演示的时候可以开启真机浏览，小程序可以直接在自己手机上或者老师手机上运行。以下是提供的一些选题，有问题欢迎提问；同时我的主页有更多项目，大家可以查看。
